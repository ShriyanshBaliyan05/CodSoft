# 🎮 Tic-Tac-Toe AI

## Overview

A command-line Tic-Tac-Toe game built in Python featuring an unbeatable AI powered by the **Minimax Algorithm**.

Unlike traditional implementations that rely on random moves, the AI evaluates all possible game states and always selects the optimal move, ensuring that it never loses.

---

## ✨ Features

* Player vs AI gameplay
* Unbeatable AI using the Minimax Algorithm
* Score tracking system
* Input validation
* Multiple rounds support
* Interactive command-line interface

---

## 🤖 How the AI Works

The AI uses the **Minimax Algorithm**, a decision-making algorithm commonly used in Artificial Intelligence and game development.

It:

* Evaluates all possible future moves
* Assumes optimal play from both sides
* Maximizes its chances of winning
* Minimizes the opponent's chances of winning

As a result, the AI will always win or force a draw.

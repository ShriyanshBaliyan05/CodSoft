# 🎬 Movie Recommendation System

A simple yet interactive **Movie Recommendation System** developed using **Python** as part of the **CodSoft Artificial Intelligence Internship**.

The program recommends movies by analyzing the genres of movies selected by the user. It identifies the user's preferred genre based on their choices and suggests similar movies from that genre.

---

## 📌 Project Overview

This project demonstrates a simple **content-based recommendation system** using genre matching. Users can select up to **five movies** from a predefined list, and the system analyzes their preferences to determine their favourite genre before recommending similar movies.

It is a beginner-friendly project that introduces the fundamental concepts of recommendation systems and user preference analysis.

---

## ✨ Features

- 🎥 Interactive console-based interface
- 🍿 Collection of 30 popular Hollywood and Bollywood movies
- ❤️ Select up to 5 favourite movies
- 🧠 Automatically identifies the user's favourite genre
- 🎯 Recommends movies based on genre preference

## 📖 How It Works

1. The program displays a list of movies.
2. The user selects up to **five favourite movies** by entering their corresponding numbers.
3. The system analyzes the genres of the selected movies.
4. Each genre receives a score based on the user's selections.
5. The genre with the highest score is identified as the user's favourite.
6. Finally, the system recommends movies from that genre.

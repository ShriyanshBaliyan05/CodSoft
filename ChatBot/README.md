# 🤖 Student Support Chatbot (CodSoft Task)

A Python-based **Rule-Based Chatbot** designed to provide emotional support, motivation, and academic guidance to students. The chatbot interacts with users through predefined conversational rules and keyword-based intent detection, enabling it to recognize common feelings and respond with encouraging, supportive messages.

## 📌 Project Overview

Student Support Chatbot is a Python-based conversational assistant developed as part of the **CodSoft Python Programming Internship**. The chatbot aims to create a friendly environment where students can express their thoughts, discuss academic concerns, seek motivation, and engage in simple conversations.

The chatbot uses keyword-based intent and emotion detection to identify common topics such as stress, exams, motivation, career, friendship, projects, and programming. Based on the user's input, it provides relevant responses, encouragement, and guidance.

## ✨ Features

* 💬 Interactive text-based conversation
* 😊 Keyword-based emotion and intent detection
* 📚 Academic support and encouragement
* 🚀 Career and motivation guidance

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/ShriyanshBaliyan05/CodSoft.git
```

2. Navigate to the Chatbot project folder:

```bash
cd CodSoft/Chatbot
```

3. Run the chatbot:

```bash
python chatbot.py
```

4. Start chatting with the bot and explore its features.
